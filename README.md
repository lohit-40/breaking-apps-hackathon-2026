# Breaking Apps Hackathon Submission

This project serves as my test suite submission for the Breaking Apps Hackathon. It utilizes [Passmark](https://github.com/bug0inc/passmark), an open-source AI library for regression testing on top of Playwright.

## Setup Instructions

1. Make sure you have Node installed.
2. Clone this repository.
3. Run `npm install`
4. Add your OpenRouter API key to the `.env` file (`OPENROUTER_API_KEY=sk-or-...`)
5. Run tests via `npx playwright test --project chromium`
